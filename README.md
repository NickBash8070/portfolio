# Portfolio

Новый независимый проект портфолио-сайта.

## Локальный запуск

Откройте `index.html` в браузере.

## Структура

- `index.html` — разметка
- `styles.css` — стили
- `script.js` — анимации и динамический год в футере
- `local-history/` — локальная история правок и черновых заметок
- `temp-previews/` — временные предпросмотры и служебные файлы проверки
